# Archived SpecKit specs (reference only)

These numbered directories are the original SpecKit-format specs (the 7-file
`spec.md` / `plan.md` / `tasks.md` / … structure). They are **historical
reference only** — superseded by the current single-file Tier-3 spec format
(`specs/<feature>.md`, see CLAUDE.md §Workflow). Do not run `/speckit-*`
commands against them and do not treat them as the source of truth for current
behaviour; the code, `docs/roadmap.md`, and the single-file specs are.
