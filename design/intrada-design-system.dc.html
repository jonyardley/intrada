Design System

Paper & Score

The locked light theme for Intrada — a practice tool that should feel like a warm manuscript page, not generic SaaS dark mode. Cream surfaces, ink-like text, serif headings over a clean sans, and a single signature move: the type-coded accent bar.

Source Serif 4 + Inter
Indigo + gold type coding
Single 640px breakpoint · iOS-native

Every colour the shell draws traces to a token in IntradaColor. Warm near-blacks over cooler cream — no pure black, no pure white — so the brand indigo stays crisp on paper.

Spacing & layout

A four-step scale — like Tailwind steps, one value serves several roles. Every padding, inset and gap traces to one of these so screens can't drift on the rhythm. Genuine one-offs stay literal.

Surface & elevation

Cards sit flat on the paper — a slightly brighter fill plus a 1px hairline does the separating. A soft shadow was tried and removed: the flatter look reads more like a printed page. No glassmorphism on content; the blur survives only on navigation chrome.

Type language
