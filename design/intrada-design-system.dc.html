Design System

Paper & Score

The locked light theme for Intrada — a practice tool that should feel like a warm manuscript page, not generic SaaS dark mode. Cream surfaces, ink-like text, serif headings over a clean sans, and a single signature move: the type-coded accent bar.

Contrast & accessibility

Text that carries information must clear WCAG AA — 4.5:1 against its surface. On cream paper that leaves room for only two readable text tiers; a third, lighter grey cannot pass — so the system is explicit about which ink does what.

ink
Titles & primary text
12.6 : 1 · AAA
inkSecondary
Body + all metadata
5.3 : 1 · AA
inkFaint
Eyebrow labels only
2.9 : 1 · fails body

Rule: metadata — keys, tempos, counts, durations — uses inkSecondary. inkFaint is reserved for uppercase eyebrow labels: bold, letter-spaced and non-essential — the same licence Apple's grouped-list headers take.

Light-only, on purpose

Intrada ships one light “paper” theme and does not adapt to Dark Mode. The metaphor is a printed score on a stand — a luminous page, not a backlit panel — and the practice ritual is short and lit. The app icon still carries Dark and Tinted renders so the home screen stays native. Should night practice prove a real need, a dedicated dark “score at night” theme is the sanctioned next step rather than an automatic invert.

Dynamic Type

Every style maps to an iOS text style and scales with the reader's setting — the serif and the sans both ship as scalable styles, never fixed points. A music app skews toward older eyes and music-stand distance, so this is load-bearing, not optional.

Mapping: pageTitle → .largeTitle, cardTitle → .title3, body → .body, meta → .footnote, badge → .caption1. Rows wrap to content height rather than fixing a height, so the largest accessibility sizes never clip; tab-bar labels follow the system and drop at AX sizes, leaving the icons.

Spacing & layout

A four-step 8-point scale — every padding, inset and gap is a multiple of 4 (8 / 12 / 16 / 24), matching Apple's native layout rhythm. The legacy 14px row was snapped to 16 so nothing sits off-grid. Genuine one-offs stay literal.
